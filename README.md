# furybee/homebrew-tap

Homebrew tap for [FuryBee](https://github.com/furybee) tools.

## Usage

```bash
brew install furybee/homebrew-tap/<formula>
```

For example:

```bash
brew install furybee/homebrew-tap/cli-games
```

## Available formulae

- [`cli-games`](https://github.com/furybee/cli-games) — A collection of terminal mini-games (Snake, Tetris, 2048, Wordle, and more)
- [`ssm-monitor`](https://github.com/furybee/ssm-monitor) — TUI monitor for AWS SSM-managed instances
